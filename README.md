# Air Quality Prediction System

This is a machine learning-based web application built with **Streamlit**. It predicts the concentration of **PM2.5** (Fine Particulate Matter) based on various air pollutant levels.

## Features
* **Real-time Prediction:** Enter pollutant levels to get instant PM2.5 results.
* **Health Advisory:** Categorizes the air quality (Good, Moderate, Unhealthy, etc.) and provides color-coded feedback.
* **Machine Learning:** Powered by a Random Forest Regressor model trained on historical air quality data.

## 🛠️ Tech Stack
* **Python**: Core programming language.
* **Streamlit**: For the web interface.
* **Scikit-Learn**: For the machine learning model.
* **Pandas**: For data handling.
